流光字体渐变效果

发布时间：2017-09-06     编辑：主小睿

这是一个纯CSS的百度学院小demo，只有短短几行代码，做出来的效果却不落窠臼，主要是利用了背景渐变的原理，调整背景大小，再让它动起来就差不多啦。

老规矩，先看demo。

流光字体渐变效果

你是大笨蛋

原理不难，重要有几个小的技术点。

1.图片模糊运用的是-webkit-filter属性。

2.伪元素的应用，初始宽度为0，left：50%，最后宽度为100%，left：0，这样就能保证线条是从中间开始加载。

3.背景渐变的运用，尤其是background-clip: text;这一步是关键，然后配合着css3关键帧动画，实现背景颜色的流动。

OK,又到了上代码的时候了。

						

							
							

							
你是大笨蛋

当当当当，大CSS粉墨登场↖(^ω^)↗

						.imgg{width: 100%;margin:0 auto;max-width:600px;position: relative;}
						.imgg img{width: 100%;}
						.imgg:hover img{-webkit-filter:blur(1px);}
						.imgg .maskk{width: 80%;height: 80%;position: absolute;left:50%;top:50%;margin-left:-40%;margin-top:-27%;}
						.imgg:hover .maskk{background-color: #000;opacity:0.2;transition:2s ease;}
						.imgg h2{display: flex;justify-content:center;align-items:center;width:80%;height:80%;position: absolute;left:50%;top:50%;margin-left:-40%;margin-top:-27%;text-align:center;font-size:40px;background:-webkit-linear-gradient(0deg, red 0%, yellow 25%, blue 50%, green 75%, red 100%);-webkit-background-size:200% 100%;background-size:200% 100%;-webkit-background-clip: text;color:transparent;-webkit-animation:5s move infinite; opacity: 0;}
						.imgg h2::before{content:"";position: absolute;width: 0;height: 100%;border-width:0;border-top-width:thick;border-bottom-width:thick;left:50%;top:0;border-color:#000;}
						.imgg:hover h2::before{width: 100%;height: 100%px;border-color:white;border-style:solid;left:0;top:0;border-width:0;border-top-width:thick;border-bottom-width:thick;transition:2s ease-in;}
						.imgg h2::after{content:"";position: absolute;width: 100%;height: 0px;border-width:0;border-left-width:thick;border-right-width:thick;left:0;top:50%;border-color:#000;}
						.imgg:hover h2::after{width: 100%;height: 100%;border-color:white;border-style:solid;left:0;top:0;border-width:0;border-left-width:thick;border-right-width:thick;transition:2s ease-in;}
						.imgg:hover h2{opacity:1;}
						@-webkit-keyframes move{0%{background-position: 0 0;}100%{background-position: -200% 0;}}

好啦，到此就基本齐活了。不过呢，由于我这边是想做响应式的，遇到了一个问题就是如何把文字设置垂直居中？！

由于h2本身的高度是随着浏览器大小而变化的，所以直接用line-height来设置是行不通的，然后我想着是不是可以用display：table以及display：table-cell来设置呢，结果尝试以后也不行，最终想到了用弹性盒模型，flexbox来设置。

						h2{display: flex;justify-content:center;align-items:center;}
						//第二种方法
						h2{display:flex;text-align:center;}
						h2 span{margin:auto;}
